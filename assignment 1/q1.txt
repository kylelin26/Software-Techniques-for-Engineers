a) 
Horner’s rule runs in O(n) time because the loop executes an n amount of times, and each iteration performs a constant amount of work. This makes the algorithm efficient for a polynomial evaluation.

b)
x = 0
for i = 0 to n:
    term = ai
    for j = 0 to i-1:
        term = term * y
    end for
    x = x + term
end for

c)
The loop invariant is that at the start of each iteration, y = Σk=0^(n-(i+1))akx^(k+i+1). We can show that the loop invariant holds at the beginning.

At the start of the first iteration, i is equal to n and y is equal to 0, so y = Σk=0^(n-(i+1))akx^(k+i+1) = 0, which is the desired loop invariant.
With an iteration where the loop invariant holds. At the start, y = Σk=0^(n-(i+1))akx^(k+i+1). After the assignment y = ai + xy, y = ai + x * Σk=0^(n-(i+1))akx^(k+i+1) = Σk=0^(n-(i))akx^(k+i). This is still equal to Σk=0^(n-(i+1))akx^(k+i+1), so the loop invariant holds for the next iteration.

With induction, the loop invariant holds for every single iterations, so at the end, y = Σk=0^(n-(i+1))akx^(k+i+1) = Σk=0^nakx^k, will be equaled to the desired result.
